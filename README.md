# Job Portal Project

A Full Stack Job Portal Web Application built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features

- User Registration
- User Login
- Add Jobs
- View Jobs
- Apply Jobs
- My Applications
- Dark Mode
- Responsive Design

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB
